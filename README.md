# 🧠 Logic Box

A Python-based menu-driven application that demonstrates fundamental programming concepts such as loops, nested loops, control statements, user input validation, and problem-solving techniques.

---

## 📖 Project Overview

Logic Box is an educational Python project developed to strengthen programming fundamentals. The application consists of two main modules:

- Pattern Generator
- Number Analyzer

The project uses nested loops, conditional statements, the `range()` function, and control statements (`break`, `continue`, and `pass`) to solve different logical problems.

---

## 🎯 Objective

To develop a Python program that generates different patterns and performs number analysis using loops, conditional statements, and control structures. The project helps improve logical thinking, problem-solving abilities, and understanding of core Python programming concepts.


## ✨ Features

### 🔹 Pattern Generator

- Generates a Right-Angled Triangle.
- Uses nested `for` loops.
- Accepts the number of rows from the user.
- Validates user input.
- Demonstrates loop control.


### 🔹 Number Analyzer

- Accepts a start and end number.
- Identifies Odd and Even numbers.
- Calculates the sum of all numbers in the given range.
- Uses the `range()` function and `for` loop.

### 🔹 Menu-Driven Interface

- Pattern Generator
- Number Analyzer
- Exit Program

### 🔹 Error Handling

- Prevents invalid row values.
- Validates number ranges.
- Displays meaningful error messages.

- Ensuring the ending number is greater than the starting number.
